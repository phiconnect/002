# 📚 Vocab Flashcards — ฝึกคำศัพท์ English → ไทย

แอปแฟลชการ์ดสำหรับฝึกคำศัพท์ภาษาอังกฤษพร้อมคำแปลภาษาไทย เป็นเว็บล้วน (HTML/CSS/JS)
ไม่ต้องติดตั้งอะไร ไม่ต้อง build เปิดในเบราว์เซอร์ได้เลย

## ✨ ฟีเจอร์
- **แฟลชการ์ด** — โชว์คำอังกฤษ แตะ/คลิก หรือกด `Space` เพื่อพลิกดูคำแปลไทย (มีอนิเมชันพลิก 3D)
- **บอกความคืบหน้า** — ปุ่ม "✅ รู้แล้ว" / "🔁 ยังไม่แม่น" พร้อมแถบคะแนนและเปอร์เซ็นต์
- **ฝึกเฉพาะคำที่ยังไม่แม่น** — โหมดวนเฉพาะคำที่ยังจำไม่ได้
- **เลือกหมวดหมู่** — ชีวิตประจำวัน, อาหาร, การเดินทาง, อารมณ์, การงาน
- **สุ่มการ์ด** และ **เริ่มใหม่** ได้
- **เพิ่มคำศัพท์เอง** — กรอกคำอังกฤษ + คำแปลไทย แล้วบันทึกไว้ในเครื่อง
- **จำความคืบหน้า** ด้วย `localStorage` (รีเฟรชแล้วข้อมูลไม่หาย)

## 🚀 วิธีใช้งาน
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย หรือรันเซิร์ฟเวอร์ในเครื่อง:

```bash
python3 -m http.server 8000
# แล้วเปิด http://localhost:8000
```

## ⌨️ คีย์ลัด
| คีย์ | การทำงาน |
|------|----------|
| `Space` / `Enter` | พลิกการ์ด |
| `←` / `→` | การ์ดก่อนหน้า / ถัดไป |
| `K` | ทำเครื่องหมาย "รู้แล้ว" |
| `J` | ทำเครื่องหมาย "ยังไม่แม่น" |

## 📂 โครงสร้างไฟล์
```
index.html      หน้าหลัก
css/style.css   สไตล์และอนิเมชัน
js/words.js     ข้อมูลคำศัพท์ตั้งต้น (~40 คำ)
js/app.js       ตรรกะของแอป (พลิก, ให้คะแนน, บันทึก)
```

## 📝 เพิ่ม/แก้คำศัพท์ตั้งต้น
แก้ไฟล์ `js/words.js` แต่ละรายการมีรูปแบบ:
```js
{ en: "morning", th: "ตอนเช้า", pos: "n.", example: "I drink coffee every morning.", category: "ชีวิตประจำวัน" }
```
`pos` (ชนิดของคำ) และ `example` (ประโยคตัวอย่าง) ใส่หรือไม่ใส่ก็ได้
